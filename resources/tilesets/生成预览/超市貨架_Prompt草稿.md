# 超市货架 Pixel Art Prompt 草稿

## 生成规格

- **像素风格：** 128x128 像素艺术
- **外观：** 空货架（内部无商品，便于后期填充）
- **视角：** 前视图 / 侧视图 / 后视图
- **比例：** 统一高度，宽度因货架类型而异

## 各视角说明

每个货架类型建议生成3个视角：

| 视角 | 说明 | 用途 |
|------|------|------|
| 前视图 (Front) | 正面面对货架 | 主要展示角度 |
| 侧视图 (Side) | 侧面90度视角 | 通道视角 |
| 后视图 (Back) | 货架背面 | 仓库补货视角 |
