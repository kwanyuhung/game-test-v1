# 游戏画面 Prompt 草稿

## 完整超市游戏画面

```
俯视角像素艺术游戏画面，64x64像素风格，呈现一家像素风超市内部：

画面布局：
- 画面上方：大堂区域 (Lobby)，米黄色大理石地板，浅色墙壁
- 左上角：收银台/服务台区域
- 中央：十字形主通道走廊，灰色中性瓷砖地板
- 中央偏下：美食广场区域，暖棕色餐桌，若干像素顾客
- 右下角：仓库区域，深灰色混凝土地板，金属货架
- 底部：卸货区，有黄色安全线标记

装饰元素：
- 走廊两侧：盆栽植物、信息亭(kiosk)、ATM机
- 美食广场：圆桌、促销展位
- 随机分布：像素风格顾客角色 (玩家、机器人)

风格要求：
- 俯视角 (Top-down view)
- 像素艺术风格，64x64像素网格感
- 清晰边缘，无抗锯齿
- 游戏截图感，非写实
- 整体画面比例 16:9
```

---

## 瓦片细节特写 Prompt

```
64x64像素艺术瓦片，超市地板材质，无缝平铺纹理：

1. 大堂地板：大理石瓷砖，米黄色，细腻纹理，反光表面
2. 普通走廊：灰色混凝土，中性色，均匀纹理
3. 仓库地面：深灰混凝土，工业风，略微粗糙
4. 美食广场：暖棕砂岩，餐桌区域，温馨感
5. 停车场：深色沥青，颗粒感纹理
6. 洗手间：浅灰瓷砖，干净整洁

像素艺术，AAA游戏品质，清晰边缘，无抗锯齿，无缝平铺
```

---

## 装饰物特写 Prompt

```
像素艺术装饰物，超市常见设施，64x64基础尺寸：

1. ATM自动取款机：蓝白色，简洁现代，像素清晰
2. 信息亭：白色框架，亮色屏幕，触摸屏
3. 贩卖机：多彩外观，玻璃展示，零食饮料
4. 盆栽植物：绿色植物，白色花盆，室内装饰
5. 圆桌：浅橡木色，木纹纹理，俯视图
6. 促销展位：彩色横幅，宣传海报，展览台

像素艺术风格，AAA游戏品质，清晰剪影，易于辨认
```

---

## 使用说明

1. **游戏全屏画面** - 使用第一个 prompt 生成整体效果预览
2. **瓦片材质** - 使用第二个 prompt 生成可平铺的地板瓦片
3. **装饰物** - 使用第三个 prompt 生成场景物件
